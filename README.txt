DEPLOY (FRONTEND ON VERCEL, BACKEND ON RENDER):
- Push project to GitHub.
- On Vercel, import repo and set root to /frontend, build: npm run build, output: dist.
- On Render, import repo and set root to /backend, start: node server.js.

ENV vars: JWT_SECRET, STRIPE keys, DATABASE_URL, FRONTEND_URL, EMAIL settings.
